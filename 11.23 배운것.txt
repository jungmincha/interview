1.HelloWorld 를 출력하는 프로그램의 과정을 설명하시오.

class HelloWorld {
	public static void main(String[]args) {
		System.out.println("hello world")
}
}




2.아래의 명령어를 설명하시오(뭐하는건가).

-javac 자바 컴파일러로서 클래스 파일을 생성한다. 오류를 판독해준다. cpu가 알아듣는 010100(기계어)로 만들어주는 과정 javac HelloWorld.java
-java 생성한 클래스 파일을 해석 및 실행한다. java HelloWorld

3.컴파일이란 무엇인가요?

cmd 창에 원하는 결과물을 출력하는 것이다.

4.java 언어를 창시한 사람은?

제임스 고슬링

5.JDK란 무엇이며, 어디서 다운로드 받으며, OS별로 버전이 있는 까닭은?
java development kit 이다. 자바 개발도구이며 오라클 사이트에서 다운로드한다. 다른 os에서는 JDK가 실행되지 않기 때문에
OS별로 맞는 버전을 실행해야 한다. 


6.JVM 무엇의 약자이며,이란 무엇인가?

Java virtual machine이다.
 os와 프로그램사이의 호환성을 생성해주며 그에 따라 os가 달라도 자바 프로그램이 실행된다.

7. path 설정하는 이유는?

운영체제가 없으면 하드웨어에서 어플리케이션을 돌릴 수 없다. 이때 운영체제가 컴퓨터의 어떤 경로에서든 특정파일(파일 형태로 이루어진 
어플리케이션)을 인식할 수 있도록 환경변수를 등록하는 것이다.(환경변수 하지 않으면 cmd에서 java 실행 불가)

즉 자바를 사용할때 파일을 인식할 수 있도록 환경 변수를 등록(운영체제가 참조하는 변수)
